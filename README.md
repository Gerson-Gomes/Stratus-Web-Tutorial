# Stratus Web Tutor

This project is a web tutorial application built with TypeScript.

## Prerequisites

- Node.js and npm installed on your machine.

## Setup

1. **Clone the repository:**
   ```bash
   git clone <repository_url>
   cd stratus-web-tutor
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

## Building the Project

To compile the TypeScript code into JavaScript, run:

```bash
npm run build
```

The compiled files will be output to the `dist` directory.

## Running the Project

Since this is a simple static site, you can open `index.html` directly in your browser.

For a better development experience, you can use a live server extension or similarly simple HTTP server:

```bash
npx serve .
```

Then open the provided URL in your browser.
